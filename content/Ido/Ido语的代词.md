| 人称    | 主格/宾格    | 复数    | 所有格    | 复数所有格   |
| ----- | -------- | ----- | ------ | ------- |
| 我     | me       | ni 我们 | mea 我的 | nia 我们的 |
| 你     | tu（亲昵）   |       |        |         |
| 你/您   | vu       | vi    | vua    | via     |
| 他/她/它 | lu       | li    | lua    | lia     |
| 他     | ilu 或 il | ili   | ilua   | ilia    |
| 她     | elu 或 el | eli   | elua   | elia    |
| 它     | olu 或 ol | oli   | olua   | olia    |
| 非特指   | onu 或 on |       | onua   |         |

反身代词：su，即他/她/它（们）自己；加上所有格：sua，即他/她/它（们）自己的。

## 造句练习

我想吃她手里的苹果。

Me volas manjar la pomo en sua mano.
